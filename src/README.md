# HotelReservationApp
Instructions to run:
1.  Go to HotelApplication and press run
2.  Once menu displays in the terminal first select 'create an account'
3.  Then select 'Admin'
4.  once in the admin menu select 'add a room'
5.  follow the prompts and add as many rooms as you'd like
6.  then select 'see all rooms' to validate room creation
7.  select 'back to main menu'
8.  then select 'find and reserve a room', follow prompts and choose room
9.  then select see my reservations to confirm
10. Select Exit to close menu
